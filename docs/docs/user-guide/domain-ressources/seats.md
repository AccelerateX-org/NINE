# Seats

Participication on occurrences.

## Occurrences

- Activity, i.e. a course
- Date, i.e. an event
- Slot, i.e. a part of an office hour

## Subcriptions

Users are subscribing occurrences.

:::info
Subscribing is a major responsibility of the user (student).
Students are not subscribed to a course by supporting staff and / or teachers.
:::
