# Hierarchies

Organisers, its members and their responsibilities in terms of curricula and module catalogs.

## Organisers

An Institution is dived into organisers, which are departments organising activities, i.e. courses, examinations, events, office hours.

An organisation is defined by its

- short name
- name

Persons acting with an organiser are called member (of the organiser). 
In general organisers are responsible for developing and operating curricula using modules.

### Members

:::info
Membership itself is a role. A single person (=account) might be a member in diffrent organisers
(even in different institutions).
:::

Members are identified by

- short name (within the organiser)
- Name

:::info
Due to asynchron planing and operating members may not have already an account.
So there is a redundancy. As a rule: if present, the name stored in the account is taken.
:::


### Curricula

Curricula are duty. Pleae refer to domain study.

### Module Catalogs

Module catalogs are necessary. Plese refer to main study.
